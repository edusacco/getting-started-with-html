Basic Concepts

The World Wide Web (or the Web): The World Wide Web (www) is an information system of interlinked hypertext documents-it's sort of a collection of HTML documents-that are accessed via the Internet. It has also commonly become known simply as the Web. Each file on the web can be located through its uniform resource locator (or URL). For more on the world wide web see [Wikipedia].

The Internet: The Internet is a global system of interconnected computer networks that use the standard Internet protocol suite (TCP/IP) to link several billion devices worldwide.

Web Pages: Individual document pages on the World Wide Web are called web pages-there are over 30 billion pages. Web pages may contain text, images, videos, and other multimedia components, as well as web navigation features consisting of hyperlinks.

Web Browser: Web pages are accessed with a software application running on the user's computer, commonly called a web browser.

Hypertext Markup Language (HTML): HyperText Markup Language, commonly referred to as HTML, is the standard markup language used to create web pages. It is written in the form of HTML elements consisting of tags enclosed in angle brackets. HTML is made up of text content, markups-made up of things called tags, references to other documents, images, videos and links to other pages. HTML documents contain text content which describes "what you see" and markup which describes "how it looks" when, for instance, you visit a web page. For a more thorough explanation click here .

Html Tags and Elements: HTML documents are made of HTML elements. An HTML element refers to everything within a set of opening and closing tags.

When writing HTML, we tell browsers the type of each element by using HTML tags. An explanation of how this works can be found by clicking here. Examples of tags are: "b" for bold, "i" for italics, "em" for emphasis-italics, "p" for paragraph, "a" for anchor, "br" for break or void tag, "img" for images.

Tags can have one or more attributes. An attribute is a property of an HTML element.

Inline vs Block Elements: HTML elements are either inline or block. While Inline elements look linear, Block elements form an "invisible box" around the content inside of them.

Thinking Like a Programmer

To think like a programmer you must develop procedural thinking, abstract thinking, systematic debugging (the skill through which programmers identify problems and/or potential problems and then track down potential causes for those problems until they are found and fixed), technological empathy (understanding how a computer see things), and systems thinking (taking one big idea and break it down into its component pieces).
